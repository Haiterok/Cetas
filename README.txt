В папці client знаходиться весь код "front-end'у", index це сам сайт. Кнопка з палітрою змінює тему веб-сайту а кнопка нижче дозволяє відключити деякі частини сайту та слайдер та посилання на Твіттер та інстаграм

В папці server знаходиться весь код "back-end'у", для його запуску потрібен Node.js . Потрібно через термінал зайти в папку потім написати npm install (встановиться з Node.js) почекати та тоді написати node app.js
